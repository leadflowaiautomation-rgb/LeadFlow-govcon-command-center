# n8n Workflows

This folder contains all exported n8n workflows for the LeadFlow AI GovCon Command Center.

## Structure

Each workflow will be exported from n8n as a JSON file and stored here.

## Naming Convention

- WF_01_Federal_Intake_Orchestrator.json
- WF_02_Download_Solicitation_Attachments.json
- WF_03_Classify_And_Extract.json
- WF_04_Bid_Scoring.json
- WF_05_Report_Generator.json
- WF_06_Document_Processor.json
- WF_07_Proposal_Generator.json

## Notes

- Always export workflows after major updates
- Keep naming consistent
- This acts as backup + version control for your automation system
